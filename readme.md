## App Order

- Port : **3000**
- Folder : order-fe
- Install: `yarn install`
- Config Database : `config/config.js`
- Migration : `npx sequelize-cli db:migrate`
- Seeder : `npx sequelize-cli db:seed:all`
- Run : `yarn start`
